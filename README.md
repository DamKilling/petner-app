# PetLife iOS App

PetLife is a SwiftUI iOS application prototype for a pet life-cycle platform.

## Current MVP

- Christmas tree album: a festive timeline gallery for pet memories.
- Video upload: pick pet videos locally and build an upload queue.
- Pet match square: a social feed and matching corner for pet companionship.

## Project Structure

- `PetLife.xcodeproj`: Xcode project file.
- `PetLife/App`: app entry and root navigation.
- `PetLife/Features`: feature screens for the three main modules.
- `PetLife/Shared`: shared models, theme, and mock data.

## Notes

- This version is a product prototype focused on app structure and UI flow.
- The upload and social features use local mock data for now.
- A future iteration can connect Firebase, Supabase, or a custom backend.
